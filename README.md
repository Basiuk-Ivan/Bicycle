# [**Bicycle**](https://bicycle-gulp.netlify.app) - landing page with fully responsive to users on various devices.

![Bicycle](https://raw.githubusercontent.com/Basiuk-Ivan/Bicycle/main/src/img/screen.png)

Bicycle is a dynamic landing page created using Gulp, fully tailored to various devices. It impresses with its well-crafted responsive design and optimized images for top performance on all devices.
